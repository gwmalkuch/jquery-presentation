Dimension Methods:




.width() and .height() set or return the width and height of selected elements







example:   $("button").click(function(){
                  $("#div1").width(500).height(500);
                  });








Value Method:




.val() sets or returns the value attribute of the selected elements







Return the value:   $(selector).val()



Set the value:   $(selector).val(value)



Set the value using a function:   $(selector).val(function(index,currentvalue))
